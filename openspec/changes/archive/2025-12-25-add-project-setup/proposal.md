# Change: Initial Project Setup

## Why
The project requires a solid base structure before any functional development. This technical foundation will allow for fast and maintainable development.

## What Changes
- Creation of monorepo structure (api/, web/, packages/)
- AdonisJS v6 configuration in API mode
- React/Vite configuration with Tailwind CSS and Shadcn UI (Neo-Brutalism)
- Multi-container Docker Compose (API, PostgreSQL, Redis)
- Strict TypeScript configuration
- Shared ESLint/Prettier setup

## Impact
- Affected specs: `project-setup` (new capability)
- Affected code: Creation of the complete project structure
