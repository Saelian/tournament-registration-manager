# Change: User Dashboard

## Why
Users must be able to view the status of all their registrations (paid, pending, waitlist) and perform actions (unregister).

## What Changes
- "My Registrations" page accessible after login
- List of all registrations linked to the email
- Distinct visual statuses
- Navigation to actions (unregister, pay balance)

## Impact
- Affected specs: `user-dashboard` (new capability)
- Affected code: `web/src/features/dashboard/`
